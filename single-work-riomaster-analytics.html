Riomaster Analytics

Built Riomaster Analytics, a full-stack game analytics platform designed for mobile game studios. The platform provides real-time player insights, level progression analysis, session tracking, retention cohort tables, monetization dashboards, and funnel visualization — all in a single self-hosted solution. It includes a lightweight Unity SDK that game developers can integrate in minutes, along with server-side webhook support for third-party services like Adjust and AppLovin MAX. The dashboard offers a Player Explorer for deep-diving into individual player journeys, a Realtime view for monitoring live activity, and configurable filters across every page. Currently used in production across multiple published mobile games, helping drive data-informed decisions on game design, difficulty balancing, and ad monetization strategy.
